  1. Install Golang
  2. Install Git
  3. Give me a star star ⭐️
  4. Make a Fork
  5. Clone the repository using git

    6. git clone https://github.com/YOUR_USER/CLI.git

  6. Install the dependencies with:

    7. go mod tidy

  7. Run the CLI with:

    8. go run main.go --help

  8. Make a new branch:

    9. git branch -M new-feat

  9. Commit your changes with the guide CONVENTIONAL COMMITS.
  10. Make a pull request describing the new feature and if pass the tests we accept your pull request or if not pass, will explain why.
